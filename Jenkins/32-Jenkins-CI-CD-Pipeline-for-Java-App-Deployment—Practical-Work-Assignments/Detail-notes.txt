**Jenkins CI/CD Pipeline for Java App Deployment**
Conceptual Understanding (English):
Jenkins is an open-source automation server that enables developers to build, test, and deploy code automatically. In a CI/CD pipeline, Jenkins pulls code from version control (e.g., GitHub), builds it using tools like Maven, runs tests, and then deploys it to a server.

```groovy
  agent any
  stages {
    stage('Checkout') {
      steps {
        checkout scm
      }
    }
    stage('Build') {
      steps {
        sh 'mvn clean package'
      }
    }
    stage('Test') {
      steps {
        sh 'mvn test'
      }
    }
    stage('Deploy') {
      steps {

      }
    }
    stage('Notify') {
      steps {
        echo 'Build completed successfully!'
      }
    }
  }
}
```
